## 技术原理

### 推理加速的本质
```
原始：摄像头(6fps) → 模型推理(2fps) → 积压 → 卡顿
优化：摄像头(6fps) → 跳帧(2fps) → 模型推理(2fps) → 流畅
GPU：摄像头(6fps) → 模型推理(60fps) → 实时 → 完全流畅
```

### 为什么跳帧不影响检测？
- 跌倒事件持续时间：1-3秒
- 跳帧后仍能捕获：每秒2帧 × 2秒 = 4帧
- 足够检测跌倒动作

### 为什么降低分辨率不影响检测？
- 跌倒检测关注：人体姿态和运动
- 640×360分辨率已足够识别姿态
- 远距离检测精度略降（可接受）
