# 告警事件记录模板

适用场景：跌倒事件、生命体征异常、设备异常、交班摘要。

## 推荐结构

1. 事件基本信息
   - 事件时间
   - 设备或摄像头名称
   - 对象姓名或设备标识
   - 事件类型
2. 检测与证据
   - 系统检测到什么
   - 持续多久
   - 是否有截图、视频、生命体征或历史告警支撑
3. 人工复核
   - 是否已联系到人
   - 现场或视频复核结论
   - 当前对象状态
4. 处理动作
   - 已通知谁
   - 是否上门、复测、送医、继续观察
5. 后续安排
   - 下一次复核时间
   - 待完成事项

## 推荐句式

- “系统于 20:14 检测到床边疑似跌倒，复核视频后见身体快速下降并倒地，当前仍未见自行起身，已联系现场人员到场查看。”
- “20:32 检测到血氧连续低于 93%，复测后恢复至 96%，结合手指冰冷和佩戴偏松，当前更偏向测量干扰，建议继续观察。”
- “当前无法仅凭视频排除真实风险，已标记为证据不足待二次复核。”

## 写作要求

- 结论要和证据强度匹配。
- 动作要具体到“谁、何时、做什么”。
- 不要把算法输出直接写成医学诊断。
