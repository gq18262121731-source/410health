# 410health 微调配置示例

本目录保存模型微调、评估、预测、导出等流程的配置示例。页面化操作优先使用主系统中的“模型微调中心”；需要命令行批处理时，可以参考这里的 YAML 配置。

## 使用方式

1. 在 `model_tuning/models/` 下准备基础模型。
2. 在 `model_tuning/data/` 中准备训练数据，并确认 `dataset_info.json` 已注册数据集。
3. 复制一个接近目标任务的 YAML 示例。
4. 修改模型路径、数据集名称、训练输出目录和显存相关参数。
5. 在 Conda 环境中执行训练或导出命令。

## 注意事项

- `models/`、`saves/`、缓存和权重文件不会上传到 GitHub。
- 示例配置中的模型名称仅用于说明模型类型，实际路径以本机环境为准。
- 业务数据进入训练前必须先完成脱敏处理。
