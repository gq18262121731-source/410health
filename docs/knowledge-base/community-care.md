# 社区康养汇总分析要点

## 风险分层建议

- 高风险对象优先看 `SOS`、持续低血氧、极端心率和明显血压异常。
- 中风险对象重点看持续发热、血氧下降趋势、反复高心率和数据质量下降。
- 低风险对象以日常随访、佩戴依从性和设备电量维护为主。

## 社区端输出建议

- 先给出社区总体覆盖设备数、风险分布和最近时间窗范围。
- 再列出需要优先核查的设备，以及对应的主要风险标记。
- 对存在长时间数据缺口的设备，要单独提示检查网关、手环电量和佩戴状态。
- 建议把上门巡检、电话随访和医生转介分为不同优先级。

## 多设备汇总时的常见关注点

- 是否出现多台设备同时血氧下降，提示环境或群体性感染风险。
- 是否存在设备集中掉线或大面积数据间隔过长，提示采集链路故障。
- 是否有高风险设备健康分持续偏低，且伴随 `SOS` 或关键生命体征异常。
- 是否存在中风险设备数量快速增长，提示需要加密社区随访节奏。
