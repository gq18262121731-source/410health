# Time-Series Trend Interpretation

适用场景：离线健康报告需要解释一个时间段内指标是上升、下降、波动还是稳定，而不是只看单个最新值。

## 常见趋势含义

- 血氧持续下降：提示呼吸状态或测量条件需要重点复查，若同时伴随气促、乏力或无应答，应升级处置。
- 体温逐步升高：提示感染或炎症风险增加，需要结合心率和精神状态判断。
- 心率持续偏高：若与发热或焦虑活动无关，应提高关注等级。
- 健康分持续下降：提示多个异常因素叠加，适合在报告中写成“整体风险上升”。
- 指标短时间大幅波动：应先提示排查佩戴、运动、低电量或采集质量问题。

## 报告写法建议

- 把趋势写成“本时段内呈上升/下降/波动/基本稳定”。
- 趋势结论后面最好跟一条证据说明。
- 如果趋势不明确，应写“当前没有足够证据判断明显趋势”。
