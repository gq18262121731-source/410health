# 跌倒事件响应与复核指引

适用场景：摄像头跌倒检测告警、人工复核、值班转派、家属沟通。

## 目标

- 先区分“高危跌倒事件”与“需要继续观察的疑似事件”。
- 先确认人身风险，再决定是否需要现场干预、电话联系或急救转运。
- 记录复核结论，避免同一事件被反复升级或重复忽略。

## 高优先级处置条件

满足任一条件时，应优先按高危事件处理：

- 告警显示快速跌倒、倒地后长时间不起、疑似失去意识。
- 跌倒后伴随长时间静止、呼叫无回应、明显挣扎或无法自行起身。
- 跌倒发生在卫生间、床边、楼梯口、狭窄角落等高风险位置。
- 同时伴随 SOS、异常心率、低血氧、体温异常或连续多次告警。
- 目标老人本身属于高龄独居、近期已有跌倒史、骨质疏松或术后恢复期。

## 人工复核要点

人工复核时建议按以下顺序判断：

1. 先看是否真的发生“人体高度突然下降并落地”。
2. 再看倒地后是否持续静止，持续时间是否超过短暂整理姿势的合理范围。
3. 看是否存在自行恢复站立、缓慢坐起、他人介入搀扶等恢复证据。
4. 结合场景判断是否可能是误报，如捡东西、系鞋带、坐地休息、宠物或遮挡干扰。
5. 若画面不足以判断，应升级为“证据不足待复核”，不要直接写成“安全无事”。

## 处置建议

### 立即升级急救

适用于以下情况：

- 无反应、疑似意识障碍、呼吸异常。
- 头颈部重击风险高，或跌倒后明显不能移动。
- 有明显外伤、出血、剧烈疼痛、抽搐。
- 跌倒前后疑似胸痛、卒中表现、突发虚弱或昏厥。

建议动作：

- 立即联系现场照护者或社区值守人员到场。
- 同步联系家属。
- 必要时拨打急救电话并保留视频片段和截图作为事件依据。

### 快速人工到场复核

适用于以下情况：

- 告警可信度较高，但画面不足以确认伤情。
- 老人能动但起身困难，或反复尝试起身失败。
- 有跌倒史、独居、夜间无人应答等附加风险。

建议动作：

- 安排就近人员上门复核。
- 在到场前通过电话、对讲或语音设备尝试沟通。
- 到场后记录意识、疼痛、出血、活动能力和是否需要转诊。

### 保留观察并继续跟踪

适用于以下情况：

- 已确认是误报。
- 老人明确恢复正常活动，且无明显不适。
- 场景可解释，且无其他多模态风险信号叠加。

建议动作：

- 标记为误报原因或低风险事件。
- 记录复核时间、复核人和结论。
- 若短时间内同类事件频发，应转为排查摄像头角度、模型阈值或环境干扰。

## 报告写法

推荐写法：

- 写清楚事件时间、地点、模型判断、人工复核结果和下一步动作。
- 明确区分“已确认跌倒”“疑似跌倒待复核”“误报”。
- 若证据不足，不要使用“已排除风险”这类过度确定表述。

不推荐写法：

- 只写“已告警，请关注”，没有结论和后续动作。
- 把单帧姿势异常直接写成“老人跌倒受伤”。
- 在未复核情况下把模型判断当作医学结论。
