# 人工复核检查清单

适用场景：值班人员、社区工作人员、家属、护理员接到系统告警后的快速核查。

## 生命体征异常复核

- 是否为单次异常还是连续异常。
- 复测后是否仍异常。
- 是否存在佩戴不稳、刚活动、手冷、情绪波动等干扰因素。
- 是否伴随症状，如胸闷、气短、头晕、发热、意识变化。

## 跌倒/姿态异常复核

- 是否真正发生落地或明显失衡。
- 异常后是否能自行恢复。
- 当前是否有反应、能否交流、能否移动。
- 是否有疼痛、外伤、出血或受困风险。

## 设备/链路异常复核

- 是否只是单台设备异常，还是整条链路异常。
- 设备是否离线、电量是否过低。
- 摄像头是否黑屏、模糊、遮挡或掉帧。
- 最近是否有维护、重启、网络波动。

## 联系与记录

- 是否已联系家属或现场人员。
- 联系是否成功，反馈内容是什么。
- 是否需要上门复核或转医疗。
- 是否补充了截图、视频片段、复测值和处理时间。

## 结论分类

- 已确认高风险，需要立即处置。
- 已确认中低风险，需要继续观察或复测。
- 证据不足，需要二次复核。
- 已确认误报，需要记录误报原因。
