# Follow-Up Guidance For Offline Reports

离线健康报告除了风险判断，还应写出后续动作，方便家属或社区直接执行。

## 家属报告建议动作

- 继续观察：适用于轻度异常、单次异常或证据不足。
- 复测确认：适用于佩戴不稳、数据波动大、单项指标异常但缺少伴随症状。
- 联系社区或家庭成员：适用于中风险、持续异常或老人主诉不适。
- 尽快线下就医或急救：适用于 SOS、无应答、极低血氧、严重胸闷气促。

## 社区报告建议动作

- 电话随访：适用于需要快速确认状态的中风险对象。
- 上门巡查：适用于高风险、联系不上或多项异常叠加的对象。
- 转介医疗：适用于急症风险、持续恶化或明显需要专业处置的对象。
- 设备排查：适用于长时间掉线、低电量、异常波动明显的数据链路问题。

## 报告表达建议

- 家属版重点写“今晚/今天该做什么”。
- 社区版重点写“谁先处理、何时跟进、由谁接手”。
