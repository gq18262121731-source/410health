# 常见生命体征阈值参考

本项目用于离线养老监测提示，不代替正式医疗诊断。判断风险时要看趋势、持续时间和伴随症状，而不是只看单个瞬时值。

## 重点观察阈值

- 血氧持续低于 93%，应提高警惕。
- 血氧低于 90%，且伴随气促、意识差或嘴唇发紫，应尽快联系家属或急救。
- 体温高于 38.0℃，说明存在发热风险。
- 心率持续高于 110 或低于 50，需要结合活动状态和既往病史重点观察。
- 收缩压持续高于 160 或舒张压持续高于 100，应尽快复测并评估不适症状。

## 离线分析建议

- 优先说明“持续异常”还是“单次异常”。
- 若多项指标同时异常，风险应上调。
- 若设备信号不稳定，先提示复测或核查佩戴状态。
