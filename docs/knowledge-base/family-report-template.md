# Family Health Report Template

适用场景：家属查看单设备、单老人健康情况时，需要的不只是短建议，还需要可直接复用的日报、周报或照护摘要。

## 推荐结构

1. 当前概况
   - 说明最新监测时间、总体风险级别、是否存在需要立刻关注的问题。
2. 关键指标
   - 概述心率、体温、血氧、血压等重点指标的变化。
3. 风险判断
   - 解释为什么判断为低、中、高风险。
   - 要区分“持续异常”和“单次异常”。
4. 家属行动建议
   - 给出观察、复测、联系社区或就医建议。
5. 数据质量说明
   - 若存在掉线、低电量、佩戴异常或数据不足，要明确写出。

## 推荐口径

- 先说结论，再说依据，再说行动。
- 语言应稳健、清楚、便于家属执行。
- 若证据不足，应说明“建议继续观察或补充数据”。
