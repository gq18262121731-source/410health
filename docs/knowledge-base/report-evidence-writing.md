# Evidence Writing For Offline Reports

本地报告生成应强调“依据来自哪里”，避免只有结论没有证据。

## 写作要求

- 先写结论，再写支持结论的监测证据。
- 若指标异常，尽量写清是单次异常、持续异常还是多项同时异常。
- 若引用知识库建议，应说明这是离线知识库支持，不是实时外部医疗服务。
- 若存在不确定性，应写明“数据不足”“需要复测”或“需线下确认”。

## 示例表达

- “近 6 小时血氧持续下降至 92% 以下，因此风险较昨日上升。”
- “两次发热同时伴随心率偏高，建议家属今晚重点观察精神状态和呼吸情况。”
- “社区侧已有 3 台设备出现高风险提示，建议优先巡查 SOS 与低血氧对象。”
