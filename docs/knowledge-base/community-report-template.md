# Community Operations Report Template

适用场景：社区值班、交班、巡查安排、阶段汇总。

## 推荐结构

1. 总体概况
   - 设备覆盖数
   - 风险分布
   - 当前最需要关注的异常类型
2. 优先对象
   - 列出高优先级设备或老人
   - 说明优先原因，如 SOS、持续低血氧、反复发热、无应答
3. 处置建议
   - 区分电话随访、上门核查、医疗转介
4. 数据质量与链路风险
   - 说明掉线、低电量、数据缺口、网关异常等问题
5. 交班提示
   - 记录待跟进对象、时限和未完成事项

## 推荐口径

- 先总后分，方便值班人员快速浏览。
- 不要只给风险判断，要给明确的下一步动作。
- 可直接复用为社区日报、值班简报或交班记录。
