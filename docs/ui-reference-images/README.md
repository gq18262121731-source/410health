# UI Reference Images

把你收集到的参考图统一放在这个目录下，后续 `UI Designer` 和 `Frontend Engineer` 都以这里为准。

这里放的是“参考图”，不是最终设计稿。

## 推荐子目录

- `00-branding`
  中文备注：品牌调性、Logo、主视觉、配色气质参考
  绑定功能：
  - 项目 Logo
  - 项目名称展示
  - 演示首页主视觉
  - 全站配色与品牌氛围
  - 状态色基线（正常 / 关注 / 异常 / 报警）

- `01-login-register`
  中文备注：登录页、注册页、身份选择、多步注册流程参考
  绑定功能：

## 每张图的备注方式

建议每张图旁边额外保留一句备注，可写在同名 `.txt` 文件里，例如：

- `login-clean-card-01.txt`
  中文备注：登录参考图说明

备注建议写：

- 你喜欢这张图的什么
- 想借鉴的是布局、配色、图表、还是交互
- 不希望照搬的部分

## 使用规则

- 这里放的是“参考图”，不是最终设计稿
- 不要把后端接口截图、控制台截图和参考图混在一起
- 联调截图继续放 `run-logs/`
- 如果一张参考图只想借鉴局部，请在同名 `.txt` 里写清楚
- 如果一张图只适合家属端或社区端，也请在备注里写明
