# 摄像头画面卡顿 - 推理加速优化方案

## 问题分析

### 当前状态
- **跌倒检测模型**：运行在CPU上
- **处理频率**：每帧都处理（`FALL_DETECTION_PROCESS_EVERY_OVERRIDE=1`）
- **速度配置**：accuracy模式（最慢但最准确）
- **画面卡顿**：推理速度跟不上摄像头帧率

### 性能瓶颈
```
摄像头 (6fps) → 跌倒检测模型 (CPU推理慢) → 前端显示 (卡顿)
```

## 推理加速方案

### 方案1：降低处理频率（最快见效）⭐

**原理**：不是每帧都推理，跳帧处理

**优化配置**：
```env
# 每3帧处理1帧（速度提升3倍）
FALL_DETECTION_PROCESS_EVERY_OVERRIDE=3

# 或使用预设速度模式
FALL_DETECTION_SPEED_PROFILE=fast  # 每3帧处理1帧
# FALL_DETECTION_SPEED_PROFILE=balanced  # 每2帧处理1帧
# FALL_DETECTION_SPEED_PROFILE=accuracy  # 每帧都处理（当前）
```

**效果**：
- ✅ 立即生效，无需安装
- ✅ 速度提升2-3倍
- ⚠️ 检测延迟增加0.3-0.5秒（可接受）

---

### 方案3：模型量化（中等效果）⭐⭐

**原理**：将模型从FP32量化到INT8，减少计算量

#### 步骤1：量化模型

创建量化脚本`scripts/quantize_model.py`：
```python
import torch
from pathlib import Path

# 加载原始模型
model_path = "D:/Program/model/fall_detection/models/fall_detection_model.pt"
model = torch.load(model_path)

# 动态量化
quantized_model = torch.quantization.quantize_dynamic(
    model, 
    {torch.nn.Linear}, 
    dtype=torch.qint8
)

# 保存量化模型
output_path = model_path.replace(".pt", "_quantized.pt")
torch.save(quantized_model, output_path)
print(f"量化模型已保存: {output_path}")
```

#### 步骤2：使用量化模型

修改配置指向量化模型（需要修改代码支持）

**效果**：
- ✅ 速度提升2-4倍
- ✅ 模型体积减小75%
- ⚠️ 精度略有下降（1-2%）
- ⚠️ 需要修改代码

### 方案4：使用TensorRT加速（最快）⭐⭐⭐

**原理**：NVIDIA TensorRT优化引擎，专为推理优化

#### 步骤1：安装TensorRT

```bash
pip install tensorrt
pip install torch2trt
```

#### 步骤2：转换模型

```python
from torch2trt import torch2trt
import torch

# 加载PyTorch模型
model = torch.load("fall_detection_model.pt").eval().cuda()

# 创建示例输入
x = torch.ones((1, 3, 224, 224)).cuda()

# 转换为TensorRT
model_trt = torch2trt(model, [x], fp16_mode=True)

# 保存
torch.save(model_trt.state_dict(), "fall_detection_model_trt.pth")
```

**效果**：
- ✅ 速度提升20-100倍
- ✅ 延迟最低
- ⚠️ 只支持NVIDIA GPU
- ⚠️ 需要修改代码

### 方案5：降低输入分辨率（快速优化）⭐

**原理**：降低输入图像分辨率，减少计算量

**优化配置**：
```env
# 降低摄像头流分辨率
CAMERA_STREAM_WIDTH=640  # 从1024降到640
CAMERA_STREAM_JPEG_QUALITY=6  # 从8降到6

# 降低跌倒检测输入分辨率
FALL_DETECTION_FRAME_WIDTH=1280  # 从2304降到1280
FALL_DETECTION_FRAME_HEIGHT=720  # 从1296降到720
```

**效果**：
- ✅ 速度提升2-3倍
- ✅ 立即生效
- ⚠️ 检测精度略降（远距离）

### 方案6：使用ONNX Runtime（跨平台加速）⭐⭐

**原理**：将PyTorch模型转换为ONNX格式，使用优化的运行时

#### 步骤1：转换模型

```python
import torch
import torch.onnx

# 加载模型
model = torch.load("fall_detection_model.pt").eval()

# 创建示例输入
dummy_input = torch.randn(1, 3, 224, 224)

# 导出ONNX
torch.onnx.export(
    model,
    dummy_input,
    "fall_detection_model.onnx",
    export_params=True,
    opset_version=11,
    do_constant_folding=True,
    input_names=['input'],
    output_names=['output']
)
```

#### 步骤2：安装ONNX Runtime

```bash
# CPU版本
pip install onnxruntime

# GPU版本（需要CUDA）
pip install onnxruntime-gpu
```

**效果**：
- ✅ 速度提升3-5倍（CPU）
- ✅ 速度提升10-20倍（GPU）
- ✅ 跨平台兼容
- ⚠️ 需要修改代码

## 快速修复脚本

创建`scripts/optimize_inference.py`：
```python
#!/usr/bin/env python3
"""
推理加速优化脚本
"""
import torch
from pathlib import Path

def check_gpu():
    """检查GPU可用性"""
    if torch.cuda.is_available():
        print(f"✓ GPU可用: {torch.cuda.get_device_name(0)}")
        print(f"  CUDA版本: {torch.version.cuda}")
        print(f"  显存: {torch.cuda.get_device_properties(0).total_memory / 1024**3:.1f} GB")
        return True
    else:
        print("✗ GPU不可用，将使用CPU")
        return False

def check_pytorch_version():
    """检查PyTorch版本"""
    print(f"PyTorch版本: {torch.__version__}")
    if "+cu" in torch.__version__:
        print("✓ 已安装CUDA版本")
    else:
        print("⚠ 当前是CPU版本，建议安装CUDA版本以获得GPU加速")

def recommend_config():
    """推荐配置"""
    has_gpu = check_gpu()
    
    print("\n推荐配置：")
    print("="*60)
    
    if has_gpu:
        print("# GPU加速配置")
        print("MODEL_DEVICE=auto")
        print("FALL_DETECTION_PROCESS_EVERY_OVERRIDE=1  # 可以处理每帧")
        print("FALL_DETECTION_SPEED_PROFILE=accuracy")
    else:
        print("# CPU优化配置")
        print("MODEL_DEVICE=cpu")
        print("FALL_DETECTION_PROCESS_EVERY_OVERRIDE=3  # 每3帧处理1帧")
        print("FALL_DETECTION_SPEED_PROFILE=fast")
        print("CAMERA_STREAM_WIDTH=640")
        print("FALL_DETECTION_FRAME_WIDTH=1280")
        print("FALL_DETECTION_FRAME_HEIGHT=720")

if __name__ == "__main__":
    print("="*60)
    print("推理加速优化检查")
    print("="*60)
    print()
    
    check_pytorch_version()
    print()
    recommend_config()
    print()
    print("="*60)
```

运行检查：
```bash
python scripts/optimize_inference.py
```

## 故障排查

### 问题1：GPU不可用

**检查**：
```bash
nvidia-smi
python -c "import torch; print(torch.cuda.is_available())"
```

**解决**：
1. 安装NVIDIA驱动
2. 安装CUDA Toolkit
3. 安装CUDA版PyTorch

### 问题2：显存不足

**症状**：`CUDA out of memory`

**解决**：
```env
# 降低批处理大小或分辨率
FALL_DETECTION_FRAME_WIDTH=960
FALL_DETECTION_FRAME_HEIGHT=540
```

### 问题3：画面仍然卡顿

**检查**：
1. 网络带宽（摄像头到服务器）
2. CPU使用率
3. 后端日志中的推理时间
