# Vue Dashboard

## 页面职责

- `社区总览`：3 秒内看清社区风险态势、待处理对象和辅助洞察。
- `家属关怀`：先确认老人是否安全，再看趋势和今日建议。
- `关系台账`：按步骤完成注册、关系建立、设备归属和履历查看。
- `调试工具`：仅用于联调实时采集，视觉和入口都与正式页面分离。

## 状态语义

- `稳定`：指标正常，可继续常规观察。
- `需关注`：出现波动，建议跟踪。
- `高风险`：需要立即干预。
- `连接异常 / 离线`：设备流或告警流不可用，数据可能延迟。
- `未绑定`：业务归属未完成，不等同于设备故障。

## 体验改造重点

- 统一了状态色、信息层级、表单与卡片语义。
- 将正式页面与调试页边界拉开，避免污染正式体验。
- 登录、注册、空状态、实时连接状态和 AI 建议过程都补了清晰反馈。

## 本地启动

```powershell
npm.cmd install
npm.cmd run dev
```
